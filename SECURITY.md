## Reporting a vulnerability

If you believe you have found a security issue, please report it privately:

- Email: ustadept@ustadept.com
- Subject: [SECURITY] USTA Token Vulnerability Report

Please include:
- a clear description of the issue
- steps to reproduce (PoC if possible)
- impact assessment (what could go wrong)
- affected commit hash / file(s)

## Scope

- Smart contracts in `contracts/`
- Deployment / verification tooling in `scripts/`

## Disclosure

We will acknowledge receipt and work toward a fix or mitigation.
No guaranteed timeline is promised.
